Welcome to Shortyz 2.0.15

This is mostly work to address bugs that have been reported. If you are experiencing crashes, please install SendLog and email me your raw log files. This will help me diagnose and fix your problems.

Previously: 2.0.8

Shortyz has been updated based on some feedback from the user community. Much of what is new in this version are options for changing the UI behavior under Settings. These include:

  • Skill Filled Boxes: Skips over boxes with answers when you enter a new letter.
  • Scroll To Clue: Scrolls the current clue to the top of the view when in clue mode. This can make working with the D-Pad/Trackball easier on this screen. (Works like Shortyz 1)
  • No Download Messages: Prevents a notification from being sent for each new puzzle downloaded.

Other fixes:

  • "Show Errors" now switches to "Hide Errors" when toggling.
  • Taping a box in Clues list works, also the scroll position is reset when you move from clue to clue, preventing long clues from "breaking" the display.
  • When "re-downloading" a New York Times puzzle updates the answer key.
  • Fixed problem of archived puzzles being re-downloaded.
  • Also, work has been done on managing the on-screen keyboard to make it feel more natural.
