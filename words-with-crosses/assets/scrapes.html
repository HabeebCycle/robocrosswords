Puzzle Archive Scraping (Experimental)

Cru Cryptics/Kegler's Kryptics The Cru Cryptics and Kegler's Kryptics archives contain a ton of puzzles. If you enable this source, Words With Crosses will download three new puzzles from each of them every time you start a download. This is likely best used with the auto-download feature turned off.

This is an experimental feature right now. In the future, I hope to offer you the ability to set up your own custom scrapes as well, but I want to start with a limited test set.
